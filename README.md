# Basic Python Calculator

A simple, single-file Python calculator script capable of performing basic arithmetic operations. 

> *Note: This project is part of my early learning journey and will eventually be merged into my broader `Python-Beginner-Projects` repository.*

## 🚀 Features
- Addition, Subtraction, Multiplication, Division.
- Lightweight and easy to run from the command line.

## ⚙️ Usage
Run the script using Python:
```bash
python calculator.py
```
